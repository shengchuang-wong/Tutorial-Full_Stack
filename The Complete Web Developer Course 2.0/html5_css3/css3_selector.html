1. div p
This select all the p element within div

2. h3 ~ p
This select all the p element that that sibling of h3

3. h3 + p
This select only one p element after the h3

4. a[target]
This is select all the a element who have target attribute

5. a[href="https://www.google.com"]
This is select all the a element who href to the https://www.google.com

6. a[href*="google"]
This is select all the a element that its URL contains google (^= begin with, $= end with, ~= has the value between two spaces, |= has the value separated by hyphens (-))
