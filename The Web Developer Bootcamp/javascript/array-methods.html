Array Methods

assume colors is an array list

1. colors.push("indigo");
Add the value into the end of array
Return: the place of the added element instead of index

2. colors.pop();
Remove the last element of the array
Return: the value of last removed

3. colors.unshift("indigo");
Add the value into the front of array
Return: the place of the added element instead of index

4. colors.shift();
Remove the first element of the array
Return: the value of last removed

5. colors.indexOf("indigo");
Return: if the indigo exist within, return the index
Return: if the indigo not exist within, return -1

6. colors.slice(1, 3);
Return the element from index 1 to index 3 exclusively
colors.slice() to copy the entire array

7. colors.forEach(function(value){console.log(value)});
Loop through all the element using forEach
